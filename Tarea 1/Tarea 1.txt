Diferencias entre Git y Sourcetree.

Git es un sistema de control de versiones en cual nos sirve para crear
los repositorios de nuestro proyectos, para:
Respaldar informacion.
Prevenir errores.
Para hacer versiones del mismo proyecto pero para diferente finalidad.

En cambio Sourcetree es un Git GUI, una interfaz grafica que nos ayudar mas facil aun,
en como realizar los cambios de una forma mas facil y ver las ramificaciones de las versiones de una
forma mas grafica y elegante.
